# Sovereign document analysis. Local inference only.
import fastapi
from edge_compute import LocalLLM, DocumentParser

app = FastAPI(title="Sovereign Router")
llm = LocalLLM(model="llama-3-8b-instruct", hardware="apple-silicon")

@app.post("/api/v1/analyze-invoice")
async def analyze_invoice(file: UploadFile):

    # 1. Parse locally. Zero cloud exposure.
    content = await DocumentParser.extract(file)

    # 2. Route to quantized local model. PII never leaves hardware.
    redacted = llm.invoke(
        prompt="Extract totals. Mask all client PII.",
        context=content
    )

    return {"status": "success", "data": redacted}

Autonomous Agents
Custom Harnesses TIER 1 For most client work, the right answer is a purpose-built scaffold. Generic agents serve committees. A custom harness is written once for your specific workflows, runs on your infrastructure, and doesn't change behaviour when someone else pushes an update. This is what I build and what I personally run.
Hermes Agent TIER 2 The apex pre-built system for agents that write their own skills on the fly. Dominant download numbers for a reason. When a custom harness is overkill, Hermes is the answer. Pre-built skills and fragile integrations are overrated. Hermes avoids both by generating skills dynamically.
OpenClaw TIER 3 Solid open-source option with respectable security defaults. Better than most off-the-shelf tools. I can set it up and have it running for you, but for serious production work, a custom harness will outlast it every time.
Coding & Architecture
Codex TIER 1 Unmatched for raw precision and rigid syntax execution. The workhorse for deterministic, structured output.
Claude Code TIER 1 The architect. Reads between the lines, bridges instructional gaps, and designs structures a human actually needs to maintain. Both Codex and Claude Code run in parallel on serious builds.
DeepMind Agents TIER 3 Third-tier use for websites, visualisations, and lighter tasks. Does the job when the job doesn't demand more.
Model Routing & Cost Efficiency
OpenRouter Critical infrastructure for intelligent model switching. Arm up for complex reasoning, drop to cheaper tiers for formatting and extraction. The cost delta at scale is meaningful, and keeping it efficient is part of the service.
Minimax 2.7 / GLM 5.1 Competitive and cost-effective for coding assistance outside a Claude Code Max subscription. Best accessed via OpenRouter only. Running the right model for the task is the difference between a sustainable system and a runaway bill.
On model selection Skip synthetic benchmarks like LiveBench. OpenRouter's utilisation rankings are the only signal that matters, because they track what developers actually deploy in production, not what performs on curated test sets. Claude remains too expensive for raw API use on most tasks. It has been effectively in-housed with additional fees inside the subscription tier. Route accordingly.
